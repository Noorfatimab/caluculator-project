# Interactive Python Calculator with Functions

This Python program lets users enter a number and see:

- Addition (+10)
- Multiplication (*3)
- Conversion to integer

The program uses **functions** to make the code clean, reusable, and professional.

## How to Run

1. Make sure Python 3 is installed
2. Clone or download this repository
3. Run the program in your terminal or IDE:

```bash
python calculator.py
